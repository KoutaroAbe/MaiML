# MaiMLTiffMerger
## (0) システムの概要
TIFF画像を出力する計測分析工程の情報をもつMaiMLファイルと、結果の情報をマージするシステムである。


## (Ⅰ) システムの使用方法
1. MorphoSmanual.pdfを参照

## (Ⅱ)　入力データ
1. document層、protocol層が存在するMaiMLファイル
1. メタデータを持つTIFFファイル

## (Ⅲ)　システムの起動方法
1. docker desktopを起動
1. docker環境を下記コマンドで起動
    docker-compose up -d
1. webブラウザからローカルホストにアクセス
    http://127.0.0.1:8000/
