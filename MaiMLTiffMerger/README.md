# MaiMLTiffMerger
## (0) システムの概要
TIFF画像を出力する計測分析工程の情報をもつMaiMLファイルと、計測結果の情報をマージするシステムである。


## (Ⅰ) システムの使用方法
1. MorphoSmanual.pdf、MaiMLTiffMerger説明.pdfを参照

## (Ⅱ) 入力データ
1. document層、protocol層が存在するMaiMLファイル
2. メタデータを持つTIFFファイル

## (Ⅲ) システムの起動方法
1. docker desktopを起動
2. docker環境を下記コマンドで起動
    docker-compose up -d
3. webブラウザからローカルホストにアクセス
    http://127.0.0.1:8000/
